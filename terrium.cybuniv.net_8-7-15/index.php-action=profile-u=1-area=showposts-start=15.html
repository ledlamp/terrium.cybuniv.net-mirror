Development / Re: What happened to Server Settings?
« on: May 12, 2015, 09:51:50 AM »
They server settings were glitching, so we removed them temporarily for development purposes. We do, however, intend on offering some basic server settings (e.g. player count). We may re-offer the vehicle physics customization server settings for exploration game modes only, but do not count on them being offered for other game modes. We plan on creating new vehicles in the future, and this can only work if they have pre-defined physics that are not customizable in competitive gameplay.

Regards,

Eric

Development / Re: Need any help?
« on: May 12, 2015, 09:48:24 AM »
Development will continue this summer. We do not need more programmers, but if you're up for it, we want a couple professionally designed worlds to use as default maps for Terrium for our beta release this summer. Think of the best world you have seen on Mars—I am looking for something at least three times as good. It needs to have various textures, 3D models, strategic locations that complement our vehicles, etc. It also needs to be designed in a way that would allow for laser tag, CTF, and racing game modes. This offer is extended to anyone up for the challenge—if you make something we decide to use, you will receive in-game credit for the map.

Regards,
