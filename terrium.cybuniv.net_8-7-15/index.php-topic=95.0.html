Mars 2.5 Source
« on: June 07, 2013, 01:24:52 AM »
Firstly, I'd like to thank Eric for making it possible for me to post again

The source code is now here!

A few things before you attempt to run it - firstly, the multitudes of errors are all ok. Unity tells you that you can't call RandomRangeInt outside the main thread; well, you can. The other error - concerning colours - is stupid too, it finds the material colour needed then reports that it doesn't exist.

One last thing: if you choose to test-run this in the editor (or in a build) DO NOT use the "Stop Hosting" button to leave a world, always use the editor's play button (or command/control-q in an application). If you press the button, the game/editor will freeze. This is because it tries to disconnect from a master server it never connected to.

So, here's the product:

https://www.dropbox.com/s/iqpo89jpxes57s7/Mars%20Explorer%202.5%20Source.zip

Enjoy, it should run happily in Unity 4. Feel free to modify to your heart's content, and don't forget that the worlds are now cached when you load them.

Credit goes to:
- Eric, for being a fantastic project leader, and for working on the code
- _ACE_ (me!), for fixing and modifying whirld (and otherwise just coding)
- Abister, for testing the project on his Windows machine just before the release… the cache wouldn't work on Windows but for his help.
- The members of the Mars Skype group, for their support and feedback
- The Mars community, for putting up with us

If you have any questions or concerns about the code, please send me a PM, I'll be glad to help wherever I can.
