Title: Multiplayer is messed up (workaround)
Post by: Lamp on July 05, 2015, 05:22:15 PM
It is very hard to join someone's game and have the terrain work. Usually what happens is the first time it gets stuck at loading asset dependencies or something like that, then afterwards it would get stuck at initilizing game or something like that. To fix this you host a game, exit, and join again, and usually you can get in, but then the terrain doesn't work.


I found a way to get in someone's game AND have the terrain work, the first time! Here is how to do it:
  • Restart Terrium
  • Host a game
  • Exit your hosted game (you'd probably get stuck there staring at the sky anyway)
  • Join the game you want
You should be able to get in the game and the terrain should work.


Edit: I have found that this work-around doesn't always work. To make it work you should host the same world as the world of the server that you want to join. Sometimes you may host, exit, and then join the game, but it still doesn't work, but then host a game again with the same world and you will actually end up in that game, and hopefully without missing terrain.
