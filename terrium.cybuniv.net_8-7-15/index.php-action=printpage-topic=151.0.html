2. Jet should have 2 different landing gear, the one that it has right now and the old one from Mars Explorer.

3. I really think you should removed the redness of the cockpit when in Gyride.

6. The speed on ALL vehicles feels VERY weird. 400 MPH feels more like 50 MPH.

11. More music variety.

13. The turning speed is terrible. Make turning faster.

14. Gliding feels weird (the speed, turning, and things like that).

15. The Tank colors don't really go with all the other vehicle colors and/or design (in my opinion).

16. Make this game compatible with Chrome.

17. I have been reading what the other players want in this game. I want it too!

18. How will you incorporate money in this game? It's in the description.

19. Put Server Settings in!

20. The acceleration on the Buggy and Jet is really annoying. They're so slow!

21. Please make it that the player limit is more than 10. I want BIG battles! Maybe team skirmishes?

22. When a Tank gets shot (Tank Battles) they literally fly into the air.

23. When a Tank joins the game, the FPS drops from 50 to 10 in a second. Example: 2 Buggies + 1 Tank               

24. Something is wrong with this text thing... When you type a letter the next letter is deleted.

25. Console Commands?

26. Move the Messaging Console to the left like Mars Explorer's, also don't make it a pop-up.

That's all I have to say right now. I know you guys are short on programmers and/or developers in this game so don't take this like your game sucks. I like the new models and the worlds, this game can be very good once you guys fix some stuff.
